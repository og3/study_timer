## study_timerについて
9:30~15:30までの時間を素敵に過ごすためのヤツ
## 実行手順
- anaconda環境での利用を想定
- ライブラリwin10toastが必要なのでこれをcondaでインストールする
```
# anacondaを起動する
$ conda activate このスクリプトを実行したい環境
# win10toastをcondaにインストールする
$ conda install win10toast
```
